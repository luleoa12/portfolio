MakeCore

August 16, 2025
HTML CSS JS Python PostgreSQL Supabase

MakeCore is a full-stack platform designed to enable users to create, publish, and share interactive programs within a collaborative online environment. The system integrates robust authentication, program management tools, community engagement features, and a modern user interface to support both individual creators and group studios.

Developed using Flask, PostgreSQL, Supabase, and a custom HTML/CSS frontend, MakeCore provides secure account handling, structured data management, and scalable deployment on Render. Additional integrations, including MailGun and GoogleOAuth, support reliable communication and platform monitoring.

Key Features

  • Program Creation & Publishing: Tools for building programs with descriptions, thumbnails, versioning, comments, and voting.
  • Studios Collaboration: Role-based member permissions, activity tracking, commenting, and shared program management.
  • MakeJam Contest System: Public leaderboards, submissions, scoring, finalization, and a built-in contest countdown.
  • Modern UI/UX Design: Responsive layout, custom iconography, dashboard with community statistics, rating graphs, and featured content.
  • Authentication & User Management: Email verification, password reset, profile editing, notifications, and settings.
  • Infrastructure & Integrations: Supabase image storage, PostgreSQL relational schema, MailGun, Google OAuth, Render deployment, and Google Analytics.
