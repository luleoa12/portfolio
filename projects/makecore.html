MakeCore

October 13, 2025
HTML CSS JS Python PostgreSQL Supabase

Description

MakeCore is a full-stack platform for creating, managing, and showcasing user-made programs, studios, and contests. It combines clean UI/UX with structured community features, including authentication, program tools, and collaborative studio spaces.

Built with Flask, PostgreSQL, Supabase, and a custom HTML/CSS frontend, MakeCore delivers secure account handling, efficient data management, and reliable deployment on Render, with MailGun and Google OAuth supporting communication and user onboarding.

Key Features

  • Program Creation & Publishing: Tools for building programs with descriptions, thumbnails, versioning, comments, and voting.
  • Studios Collaboration: Role-based member permissions, activity tracking, commenting, and shared program management.
  • MakeJam Contest System: Public leaderboards, submissions, scoring, finalization, and a built-in contest countdown.
  • Modern UI/UX Design: Responsive layout, custom iconography, a dashboard with community statistics, rating graphs, and featured content.
  • Authentication & User Management: Email verification, password reset, profile editing, notifications, and settings.
  • Infrastructure & Integrations: Supabase image storage, a PostgreSQL relational schema, MailGun, Google OAuth, Render deployment, and Google Analytics.
